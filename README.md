# Velder-soft

Profesjonalny system zarządzania projektami inżynieryjnymi dla firmy **Velder**. Rozwiązanie wieloplatformowe działające jako aplikacja mobilna (Android/iOS) oraz panel webowy.

## 🚀 Szybki Start (Developer)

```bash
# 1. Zainstaluj zależności
npm install

# 2. Uruchom wersję Web
npm run web

# 3. Uruchom wersję mobilną (Expo Go)
npx expo start
```

_Wymaga skonfigurowanego środowiska Firebase (klucze API)._
