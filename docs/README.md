# Velder-soft: Система управління проектами

Професійний кросплатформний додаток (Mobile + Web) для компанії **Velder**, розроблений для автоматизації контролю інженерних проектів.

## 🛠 Технологічний стек

- **Frontend:** React Native (Expo) + TypeScript.
- **Web:** React Native for Web (один код для ПК та смартфона).
- **Backend (BaaS):** Firebase (Auth, Firestore, Storage).
- **Стилізація:** Styled-components (адаптивний дизайн).
- **Іконки:** Lucide-react-native.

## 🚀 Як швидко запустити

1. **Клонуйте репозиторій.**
2. **Налаштуйте оточення:**
   - Скопіюйте `.env.example` у `.env`.
   - Вставте ключі вашого Firebase проекту.
3. **Встановіть залежності:**
   ```bash
   npm install
   ```
4. **Запустіть проект:**
   - **Для Вебу (ПК):** `npm run web`
   - **Для Мобілок:** `npx expo start` (відскануйте QR-код в Expo Go).

## 🔑 Управління ролями

Додаток підтримує дві ролі: `DIRECTOR` та `EMPLOYEE`.

- **Для розробки:** Наразі перевірка ролей може бути спрощена для швидкого тестування.
- **Як стати Директором:** У колекції `users` у Firestore знайдіть документ з вашим `uid` та встановіть:
  ```json
  {
    "role": "DIRECTOR",
    "isActive": true
  }
  ```

## 📁 Структура папок

- `src/config`: Налаштування Firebase та Теми.
- `src/context`: Логіка авторизації.
- `src/screens`: Екрани додатка.
- `src/components`: Повторно використовувані компоненти (Layout та інше).
- `docs/`: Детальна документація.
